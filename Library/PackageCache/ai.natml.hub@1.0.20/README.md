# NatML Hub

![NatML](https://raw.githubusercontent.com/natmlx/.github/main/natml.png)

NatML Graph API for Unity Engine.

## Installing Hub
Add the following items to your Unity project's `Packages/manifest.json`:
```json
{
  "scopedRegistries": [
    {
      "name": "NatML",
      "url": "https://registry.npmjs.com",
      "scopes": ["ai.natml"]
    }
  ],
  "dependencies": {
    "ai.natml.hub": "1.0.20"
  }
}
```

## Requirements
- Unity 2021.2+
